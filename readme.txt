Программа обрабатывает файлы HTML, PHP, CSS и заменяет пути внутри них.

Что реализовано:
- выбор тип задачи и номер команды.
- функция walkSync рекурсивно обходит все поддиректории проекта и собирает список всех файлов.
- пропускает пути, которые начинаются с префиксов http, //, / и пути с включением известных внешних источников (например, шрифты Google, cdnjs).
- пропускает изображения в формате data:image.
- если путь начинается с ./ или ., то предполагается, что он ведет к index.html в текущей директории.
- пути, начинающиеся с #, ведут к якорным ссылкам внутри текущего index.html.
- в атрибутах href, src, xlink:href (используется в SVG) проверяются пути и преобразуются, если они не являются внешними.
- для srcset каждый ресурс обрабатывается отдельно.
- url() внутри CSS файлов
- после обработки путей, создается базовый URL в соответствии с типом задачи
- ссылки websites всегда ведут к index.html, а другие типы задач (например, папка целиком) ведут к корневой директории.
- логирования в консоли
